# APIって何？
## 目的
- APIって何？を理解

## APIって何？
- Application programming interfaceの略
- 他のサービス機能を連携できる仕組み
- 決まった方法でアクセスすると、決まった結果を返してくれる
- データの二次利用、効率的な開発などのメリット
- Facebook API, LINE API, YouTubeAPI, チャットワークAPI, Amazon API, GoogleMap APIなど多数の企業が公開している
- レスポンスがHTMLでなくJSON
